Configure which domains to monitor for LaunchDarkly feature flags
Domain Patterns *
Enter the domains you want to monitor for LaunchDarkly network requests. You can use wildcards (*) for subdomains.
Use *.example.com to match all subdomains or app.example.com for specific domains.
Examples:
*.example.com - Matches all example.com subdomains
app.example.com - Matches only app.example.com
*.staging.example.com - Matches all staging subdomains
