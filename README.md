# LaunchDarkly Inspector

A Chrome extension that captures and displays LaunchDarkly feature flags from configured domains.

## Features

- Captures feature flags from network requests on domains you configure
- Searchable popup interface with flag keys and values
- Direct links to the LaunchDarkly dashboard
- All data stored locally in your browser

## Installation

1. Download the latest version from the [Tags page](https://github.com/sebkolind/chrome-ld-inspector/tags)
2. Extract the zip file
3. Navigate to `chrome://extensions/`
4. Enable "Developer mode" (toggle in top right)
5. Click "Load unpacked"
6. Select the extracted directory

## Configuration

- **Enable domains** - Click to allow/deny domains for monitoring
- **Project key** - Set your LaunchDarkly project key in the popup for dashboard links

Find your project key at: https://app.launchdarkly.com/settings/projects
